# Fitness Analysis Web Application
A full-stack fitness analysis web application built using HTML, CSS, JavaScript, and Python (Flask).

## Live Demo
🔗 https://fitness-analysis-web-app.onrender.com

## Overview
This application analyzes user fitness data and provides personalized insights using scientific fitness calculations and interactive visualizations.
Users can enter details such as age, gender, height, weight, activity level, fitness goal, workout frequency, and daily steps to receive a complete fitness assessment.

## Features
- BMI Calculation
- BMR Calculation (Mifflin-St Jeor Equation)
- Daily Calorie Requirement Estimation
- Overall Fitness Score (0–100)
- Personalized Fitness Recommendations
- Animated Circular Fitness Score Meter
- BMI Range Visualization
- Population Comparison
- Gym-Themed Dark User Interface
- Responsive Design
